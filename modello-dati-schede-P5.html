Lista delle regole applicate alla scheda.

Regole

Codice Descrizione Test
REG98 L'importo indicato nell'oggetto appalto.datiBase.importo deve essere > 0. appalto.datiBase.importo = null or appalto.datiBase.importo <= 0
REG99 L'importo indicato nell'oggetto lotto.datiBase.importo deve essere > 0. some lotto in lotti satisfies lotto.datiBase.importo = null or lotto.datiBase.importo <= 0
REG110 La somma degli importi indicati nei lotti deve essere compresa tra l'80% e il 100% dell'importo indicato nell'appalto. In presenza di eForm verificare gli importi inseriti nei campi EstimatedOverallContractAmount (BT-27). (sum(for i in 1..count(lotti) return lotti[i].datiBase.importo) < (appalto.datiBase.importo * 0.8)) or (sum(for i in 1..count(lotti) return lotti[i].datiBase.importo) > appalto.datiBase.importo)
REG101 indicare almeno uno dei campi di cui si richiede la modifica some lotto in lotti satisfies count(lotto.cup)>1 and (every c in lotto.cup satisfies (c = null or c = ""))
