Regole

Codice Descrizione Test
REG114 Indicare l'aggiudicatario. count(aggiudicazioni) <= 0
REG114_1 Deve essere indicato un solo aggiudicatario. count(aggiudicazioni) > 1
REG115 Indicare una sola categoria di tipo prevalente some aggiudicazione in aggiudicazioni satisfies (count(aggiudicazione.categoriaOC[item.tipo = "PREVALENTE"])=0 or count(aggiudicazione.categoriaOC[item.tipo = "PREVALENTE"])>1)
REG116 Indicare una sola cpv di tipo prevalente some aggiudicazione in aggiudicazioni satisfies (count(aggiudicazione.cpvOC[item.tipo = "PREVALENTE"])=0 or count(aggiudicazione.cpvOC[item.tipo = "PREVALENTE"])>1)
REG123 E’ obbligatorio indicare i cup associati al lotto se l’appalto o concessione è afferente agli investimenti pubblici finanziati con le risorse previste dal PNRR e/o PNC some aggiudicazione in aggiudicazioni satisfies aggiudicazione.afferenteInvestimentiPNRR = true and (is defined(aggiudicazione.cupLotto)=false or count(aggiudicazione.cupLotto) = 0)
REG124 Indicare almeno uno tra codice istat e codice nuts. some aggiudicazione in aggiudicazioni satisfies aggiudicazione.codIstat = "" and aggiudicazione.datiBaseContratto.codNUTS.codice = ""
REG125 Indicare solo uno tra codice istat e codice nuts. some aggiudicazione in aggiudicazioni satisfies aggiudicazione.codIstat != "" and aggiudicazione.datiBaseContratto.codNUTS.codice != ""
